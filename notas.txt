Datos Primitivos
- String -> todo valor considerado texto, ""
- Number -> Todo valor considerado un numero 
- Boolean -> Todo valor logico (True o False)
- Null -> Todo valor sin determinar (no tiene valor)
- Undefinded -> Todo valor sin inicializar
